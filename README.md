# The Midnight Press - Luxury Café & Reservation Platform ☕🌌

Welcome to **The Midnight Press**, a premium, high-end dark industrial café website. This platform combines an immersive user-facing experience with an upcoming table reservation engine and an authorized staff administration dashboard.

## ✨ Features Included

### 🌙 Customer-Facing Site
* **Premium Industrial Aesthetic:** Dark-mode themes paired with luxury copper accents, typography, and interactive map displays.
* **Interactive Table Reservations:** A validation-ready modal form designed to collect client info: *Name, Phone, Date, Time, and Guest counts*.

### 📊 Authorized Admin Dashboard View
* **Staff-Only Analytics:** Quick-view metric cards tracking total daily bookings, pending slots, and expected guest volumes.
* **Interactive Data Matrix:** Comprehensive management table displaying incoming customer reservations with operational controls like *"Check In"* or *"Cancel Booking"*.
